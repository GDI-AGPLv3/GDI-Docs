# Memo Recibido

Cuando abris un memo que recibiste, el sistema registra automaticamente la apertura. Esta informacion es visible para el remitente.

---

## Vista del receptor

### Primera apertura

La primera vez que abris un memo recibido, se muestra un badge **"Primera vez que abres este memo"**. A partir de ese momento, el remitente puede ver que lo abriste y cuando.

### Contenido

- **PDF del memo** firmado
- **Informacion del remitente**: nombre completo y sector
- **Fecha de firma**

### Panel de destinatarios

Ves los destinatarios del memo:

| Grupo | Que ves |
|-------|---------|
| **Para (TO)** | Nombre completo y sector |
| **CC** | Nombre completo y sector |
| **CCO (BCC)** | No visible (solo el remitente los ve) |

### Resumen IA

Si el sistema genero un resumen automatico, se muestra para lectura rapida.

---

## Archivar un memo

Para mantener limpia tu bandeja de recibidos, podes archivar memos:

1. Abri el memo
2. Presiona el boton **"Archivar"**
3. El memo se mueve a la pestana **Archivados**

### Desarchivar

Si necesitas volver a ver un memo archivado en tu bandeja:

1. Anda a la pestana **Archivados**
2. Abri el memo
3. Presiona **"Desarchivar"**
4. El memo vuelve a **Recibidos**

!!! info "Archivar es personal"
    Archivar un memo solo lo mueve de TU bandeja. Los demas destinatarios siguen viendolo en sus bandejas normalmente. El remitente no se entera de que lo archivaste.

!!! info "Solo destinatarios pueden archivar"
    El remitente no puede archivar sus propios memos enviados. Solo los destinatarios (TO, CC, CCO) pueden hacerlo.
